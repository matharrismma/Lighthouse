# The wedge

The one move: **be the verification floor for AI agents, and let one stunning, checkable connection be the thing humans forward.** The shareable artifact lives at [`site/ten-second-proof.html`](../site/ten-second-proof.html). This is the agent-facing pitch.

## For agents and the developers who run them

> Your model hallucinates numbers. This one verifier doesn't.
>
> Narrow Highway is an MCP server that **checks** a derivation deterministically — across ~66 domains (math, physics, chemistry, statistics, biology, CS…) — and returns HOLDS / BROKEN with a tamper-evident, citable seal. It never generates an answer; it judges the one you give it. Wrong formalization → BROKEN. No hallucination surface.
>
> Connect it: `https://narrowhighway.com/mcp`
> Or call it directly: `POST https://narrowhighway.com/derivation/verify`

**Why an agent uses it:** verification is the one thing a language model cannot do for itself. An agent that wants to stop shipping confident-but-wrong math wires this in as its floor. Each gap it closes becomes a deterministic rule, so the engine's dependence on any LLM *falls with use* — it gets more trustworthy, not less.

**Ten-second example.** Ask it to verify "a moving clock runs slow by the 3-4-5 ratio (time dilation from Pythagoras)." It returns HOLDS with a seal you can open and re-check: `https://narrowhighway.com/seal/4da00f9a1dc0c7002ee3bab06b63c847f1983c4a670f2d7b5fe38e42a76f5802`. Try to slip it a false step and it returns BROKEN with the true value. The trail *is* the reasoning.

## What makes it shareable (human side)

Verifiable claims forward themselves: *"don't trust me — check the seal."* The hook is a connection that surprises (time dilation is a triangle; pH and decibels are one logarithm) wrapped around a receipt anyone can open. Delight (a true surprise) plus relief (something that doesn't lie). That is the tell-a-friend artifact, and it is the same un-copyable corpus that is the moat.

Two shareable surfaces exist: [`site/ten-second-proof.html`](../site/ten-second-proof.html) (one stunning proof in ten seconds) and [`site/proof-bridges.html`](../site/proof-bridges.html) — the **growing wall of word-to-form bridges**, seventeen teachings of Christ so far, each bonded to a verified mathematical form with its own live `/seal` link. That second page is the deepest, most un-copyable thing the project shows: anyone has commentary, anyone has math, no one ties them with a receipt underneath. The center of it — the join where the Word and the form are one (Col 1:17) — is deliberately left open, the one stone reserved for the operator.

## What this is not

Not a chatbot, not a search engine, not another AI that answers. It is the thing that tells you whether an answer is true — and proves it. The rest of the site (codex, almanac, games, tools) is the gift you give once someone arrives; this is the wedge that gets them there.
