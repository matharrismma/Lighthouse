Connect any agent in 60 seconds

Four ways in. The engine is the same. The interface is yours.

Add this to your claude_desktop_config.json — narrowhighway.com hosts the MCP server, so you don't install anything.

// ~/Library/Application Support/Claude/claude_desktop_config.json (mac)
// %APPDATA%/Claude/claude_desktop_config.json (Windows)
{
  "mcpServers": {
    "concordance": {
      "url": "https://narrowhighway.com/mcp"
    }
  }
}

The HTTP REST API exposes 86 endpoints — capture, verify, seal, journal, ledger, polymathic synthesis. No auth needed for read endpoints.

# Verify a chemistry claim
curl -X POST https://narrowhighway.com/verify/chemistry \
  -H "Content-Type: application/json" \
  -d '{"equation":"H2 + O2 -> H2O"}'

# Browse the live verifier inventory
curl https://narrowhighway.com/verifiers
curl https://narrowhighway.com/llms.txt

Four gates. Fixed order. First failure halts.

Every packet — claim, equation, decision, verse — flows through the same path. The engine never self-confirms.

01 · Red

Coercion check

Rejects unilateral authority, rights violations, hard computational failures. The first thing kept out of the ledger.

02 · Floor

Structural floor

Rejects incomplete or internally inconsistent packets. Required fields per domain. Scripture-anchor coverage.

03 · Brothers

Witness threshold

Quarantines if witnesses are insufficient or the review window has not elapsed. Two or three witnesses establish a matter (Deut 19:15).

04 · God

Permanent record

Seals into the append-only audit chain. Every step of the reasoning trail is captured. Nothing the engine cannot show.
