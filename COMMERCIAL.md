# Commercial licensing

This project is given freely. Almost everyone can use all of it, forever, at no
cost. This page is only for the narrow case of a company that wants to use it in
a way the open licenses don't allow -- and it explains why paying, in that case,
is part of the mission rather than a contradiction of it.

## Free, always -- no license needed

You never need a commercial license, and you never pay, if you are:

- an **individual** or a **family**;
- a **church**, ministry, **nonprofit**, school, or community group;
- a person or group **serving the poor, the sick, the hungry, or the least**;
- an **AI agent or developer using the hosted API / MCP** at narrowhighway.com
  to serve real people;
- anyone willing to honor the open licenses: **AGPL-3.0** for the code and
  **CC BY-SA 4.0** for the content (use it, modify it, run it, share it -- and
  if you modify and host it, publish your changes and keep it under the same
  terms).

If that's you, stop reading. It's yours. Go in peace.

## When a company needs a commercial license

You need a commercial license **only** if your company wants to use this project
*without* the share-alike obligations -- for example to:

- embed the engine inside a **closed-source or proprietary product**;
- run a **modified version as a hosted service** without publishing your source;
- redistribute the **content** without the CC BY-SA attribution / share-alike
  terms;
- otherwise **enclose** what was given freely.

The open licenses already let you *use* the project commercially for free, as
long as you keep it open. The commercial license exists for the case where you
want to keep your derivative closed. In that case, the principle is simple: if
you are going to make money from it, you contribute, and that contribution funds
the giving.

## The terms (in spirit)

- A **fair, proportionate contribution** -- enough to sustain the work and fund
  the mission, not a pound of flesh. We are not trying to maximize revenue.
- **Every commercial dollar is earmarked for giving first**, and the giving is
  verified -- you can check where it goes, by the same conservation discipline
  the engine applies to everything else. You are not paying into a profit
  center; you are paying into a transparent giving machine.
- Third-party components keep their own upstream licenses; a commercial license
  here covers this project's own code and authored content, not the external
  data in `lw/00_source/` (which carries its own terms -- see `NOTICE` and
  `docs/DATA_SOURCES.md`).
