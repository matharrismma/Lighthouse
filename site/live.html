Add Narrow Highway to your apps

Plex Live TV (custom IPTV)

  1. Open Plex → Settings → Manage → Live TV & DVR.
  2. Click Set up Plex DVR → Continue without a tuner → Have an M3U tuner.
  3. Paste the URL: https://narrowhighway.com/channels/narrow-highway/hls/day.m3u8
  4. For EPG (program guide), use: https://narrowhighway.com/channels/narrow-highway/epg.xml
  5. Save. The channel appears in Plex's Live TV guide.

Jellyfin Live TV

  1. Open Jellyfin admin → Dashboard → Live TV → Add.
  2. Tuner type: M3U Tuner. URL: https://narrowhighway.com/channels/narrow-highway/hls/day.m3u8
  3. Save the tuner, then add a guide source: type XMLTV, URL: https://narrowhighway.com/channels/narrow-highway/epg.xml
  4. Refresh the guide. Narrow Highway appears in your Live TV section.

VLC (any platform)

  1. Open VLC → Media → Open Network Stream (Ctrl+N / Cmd+N).
  2. Paste: https://narrowhighway.com/channels/narrow-highway/hls/day.m3u8
  3. Click Play.

Kodi (PVR IPTV Simple Client)

  1. Install the PVR IPTV Simple Client add-on (Add-ons → My add-ons → PVR clients).
  2. Configure → M3U Play List URL: https://narrowhighway.com/channels/narrow-highway/hls/day.m3u8
  3. EPG: XMLTV URL: https://narrowhighway.com/channels/narrow-highway/epg.xml
  4. Enable the add-on and restart Kodi. Channel appears under TV.

TVHeadend / generic IPTV

  1. Add a new IPTV network or playlist source.
  2. URL: https://narrowhighway.com/channels/narrow-highway/hls/day.m3u8
  3. EPG (XMLTV): https://narrowhighway.com/channels/narrow-highway/epg.xml
