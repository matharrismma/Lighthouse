# Computer Science Canon v1.0 Structure

This version integrates the Computer Science Canon v1.0 into the Lighthouse repository with the following structure:

/computer_science/
  ├── core/
  ├── templates/
  ├── modules/
  ├── schema/
  └── tools/

## Overview
This documentation provides an overview of how the Canon is organized within the repository, and the purpose of each subdirectory.

- **core/**: Contains fundamental concepts and theories of computer science.
- **templates/**: Various templates related to computer science practices.
- **modules/**: Separate modules covering specific topics in computer science.
- **schema/**: Defines data structures relevant to computer science.
- **tools/**: Tools and resources to aid in computer science applications.

## Accessing v1.0
To access version 1.0 of the Computer Science Canon, navigate to the `/computer_science/` directory in the project root.
