# The Review of Every Teaching of Christ — the map

**Matt 2026-06-12:** *"Do a full review of every teaching of Christ"* — Phase 3, the language tree (the Tree of Life, the Word). And: *"do a full review of the project; don't miss work we've done in this area."* This file is the integration map: what already exists (build on it), the method, and the real gaps.

## The method (the language-tree review)
For each teaching, record four things, nested into the Tree of Life → the codex → the one ground (Christ):
1. **Original Greek + word-parts** — the frozen anchor (only Scripture is frozen, at the word-parts). Use the live engine: `GET /scripture/original?ref=` (MorphGNT + Strong's), `word_study(G####)`, `triangulate_claim`.
2. **The teaching plainly** — it means what it says (original intent).
3. **Its wisdom on every axis** — the map's 7 axes / the domains (Matt: "applied to every axis").
4. **Its nesting** — to the existing structure, not a parallel one. The join (Col 1:17) is reserved for the operator.

## What ALREADY EXISTS (do not duplicate — integrate)
- **Sermon on the Mount:** `data/sermon_on_mount/units.jsonl` — all 20 units (som_01..som_20) with WEB text + axes + themes. **The spine.** The review ENRICHES each unit with Greek word-parts + per-saying wisdom; it does not recreate the text.
- **FieldKit:** `data/fieldkit/v1_cards.jsonl` — 13 cards (7-day practices, floor/spice/drift) mapping to 12 SOM units.
- **Saying cards** (corpus): `salt_earth` (som_02), `narrow_road`/`narrow_gate_few` (som_17), `build_on_rock` (som_20), `judge_not`+`plank_in_eye` (som_15), `take_up_cross` — verified, axed. **Bond each to its unit.**
