# Scope
# narrowhighway.com and any subdomain pointing to the Concordance
# engine. The Cloudflare tunnel terminates TLS; the origin runs
# behind a residential ISP and is not part of any bug-bounty program.
#
# Out of scope
# - Reports about missing rate limits on read-only endpoints
#   (those are deliberately open).
# - Reports that depend on physical access to the operator's machine.
# - Reports based purely on theoretical concerns about the engine's
#   doctrinal positions (different category — open an issue instead).
#
# In scope
# - Path traversal, SSRF, RCE, auth bypass, ledger tampering that
#   passes /ledger/verify, signature forgery on /packets/*/verify-sig,
#   cost-runup attacks that bypass /visits and rate limits.
