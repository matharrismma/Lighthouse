## 1. Data Source

- **NHANES 2011–2018** (4 consecutive 2-year cycles: G, H, I, J)
- **Optional:** NCHS Public-Use Linked Mortality File (LMF)

## 2. Layer Proxy Definitions

All continuous indicators are z-scored **within each 2-year cycle** to
prevent batch effects. Each layer composite = **mean of observed z-scores**
(not sum) among indicators that pass QC.

### Minimum-Observed Gating

A participant's layer composite is set to **NaN** (not zero, not deflated)
unless they have at least the minimum number of valid (non-missing,
non-special-code) indicators for that layer.

| Layer | Indicators | Min Obs |
|-------|-----------|---------|
| **L1 proxy** (cellular stress) | LBXRDW, LBXSGTSI, LBXSALT, LBXSUA, LBXFER | 3 of 5 |
| L2 (Metabolic) | LBXGLU, LBXGH, BMXBMI, BMXWAIST, TG_HDL_ratio | 3 of 5 |
| L3 (Immune) | LBXCRP, LBXWBCSI | 2 of 2 |
| L4 (Tissue) | SBP_mean, DBP_mean, LBXSATSI, LBXSASSI, LBXSCR, LBXSAL | 3 of 6 |
| L5 (Systemic) | BPXPLS, short_sleep, depression_score, smoker_ever | 2 of 4 |
| Upstream Manifold | L5_systemic, L3_immune, L2_metabolic | 2 of 3 |

> **L1 proxy scope limitation:** NHANES does not contain direct L1 biomarkers
> (HSP70/72, γ-H2AX, mitophagy flux, UPR markers). The L1 proxy composite uses
> *downstream consequences* of cellular stress: RDW (oxidative RBC damage), GGT
> (glutathione cycling), ALT (hepatocellular damage), uric acid (purine catabolism),
> and ferritin (iron-mediated oxidative stress). This is generated by the E3 extension
> module and does **not** test the framework's mechanistic L1 claims. True L1
> validation requires a molecular cohort with proteomics (e.g., UK Biobank Olink). See EXTENSIONS_PLAN E4.

### Upstream Manifold

`upstream_manifold = mean(z(L5), z(L3), z(L2))` where z() is cycle-wise
standardization of the layer composite itself, gated by min-obs ≥ 2 of 3.

## 4. Hypothesis Operational Definitions

### H1 — Shared Manifold Predicts 10-Year Mortality

- **Cohort:** Adult participants with non-missing mortstat, permth_int,
  RIDAGEYR, RIAGENDR, and upstream_manifold (gated)
- **Outcome:** Binary; 1 if mortstat==1 AND permth_int ≤ 120, else 0
- **Baseline model:** Logistic regression on age + sex
- **Test model:** Logistic regression on age + sex + upstream_manifold
- **Evaluation:** 5-fold stratified CV, metric = AUC
- **Falsifier:** AUC_test < 0.60 **OR** ΔAUC ≤ 0 vs baseline
- **Minimum requirements:** N ≥ 500 eligible, events ≥ 50

### H2 — Multi-Path Hyperglycemia Heterogeneity

- **Band definition:** Fasting glucose ≥ 100 mg/dL **OR** HbA1c ≥ 5.7%
- **Feature set:** LBXCRP, SBP_mean, DBP_mean, LBXSATSI, LBXSASSI, BMXBMI
  (all non-missing after QC recoding)
- **Method:** Gaussian Mixture Model, k ∈ {2,3,4,5}, BIC selection on
  training split (70%)
- **External validation (test split 30%):**
  Chi-square test of cluster × medication (insulin, diabetes pills)
- **Stability requirement:** ARI ≥ 0.40 across 50 bootstrap resamples
- **Cycle consistency:** Cluster profiles must not be driven by a single cycle
- **Falsifier:** best_k < 2 **OR** medication p > 0.05 in test split
  **OR** ARI < 0.40 **OR** cycle-inconsistent

### H3 — Layer 5 Adds Predictive Lift for Diabetes

- **Outcome:** Lab-defined diabetes (fasting glucose ≥ 126 OR HbA1c ≥ 6.5%)
- **Base model:** Logistic regression on age + sex + L2 + L3
- **Test model:** Logistic regression on age + sex + L2 + L3 + L5
- **Evaluation:** 5-fold stratified CV, metric = AUC
- **Falsifier:** ΔAUC_L5 < 0.015 across pooled CV **OR** negative in any
  cycle with adequate N (≥ 300 with ≥ 2 classes)
- **Clinically meaningful threshold:** ΔAUC ≥ 0.02 (required for SUPPORTED)
- **Additional metrics:** Brier score delta (calibration), partial AUC at
  max_fpr=0.2 (high-specificity discrimination)

## 6. Survey Weight Handling

- **Primary analysis (prereg mode):** Unweighted. This is locked.
- **Sensitivity mode:** Combined 8-year weights.
  - MEC weight for exam-based analyses: `WTMEC8YR = WTMEC2YR / 4`
  - Fasting subsample weight where applicable: `WTSAF8YR = WTSAF2YR / 4`
  - Weighted AUC via `roc_auc_score(sample_weight=...)`
  - Weighted logistic regression via `sample_weight=...`
- A conclusion is **stable** if the sensitivity mode does not reverse
  the primary verdict's direction of effect.
