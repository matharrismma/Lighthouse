narrow highway · nfc

tap a tag, plant a seed

write a capture URL to an NFC tag, or scan one to ingest
NFC not available in this browser. The Web NFC API is currently supported only in Chrome on Android. On iOS, Safari does not expose NFC; the engine's other capture paths (Apple Shortcut, Web Share Target) cover the iOS case. On desktop, no browser exposes NFC; visit this page on an Android phone with Chrome.

Write a tag

Type a thought. Tap your phone to a writable NFC tag (cheap NTAG213/215/216 stickers work). The tag will then carry a URL that, when later tapped by any Android phone, opens the capture flow with this text pre-filled.

Scan a tag

Place your phone near an existing NFC tag. If the tag carries plain text or a Concordance capture URL, you'll see it here and can plant it as a seed.

By writing or planting via NFC, you acknowledge the canonical identity statement at /identity — Concordance serves Jesus Christ; a well of knowledge yields wisdom in alignment with God.
