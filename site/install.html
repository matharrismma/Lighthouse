Setup · 60 seconds

Install

Three minutes from a fresh machine to verifier tools showing up in Claude Desktop's tool picker.

Prerequisites

  • Python ≥ 3.10
  • git
  • (For MCP integration) Claude Desktop, Claude Code, Cursor, or another MCP-capable client

1. Clone and install

git clone https://github.com/matharrismma/Lighthouse
cd Lighthouse
python -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install -e ".[mcp]"

That installs the engine, scientific dependencies (sympy, numpy, scipy), and the MCP SDK that powers the server.

2. Confirm it works

PYTHONPATH=src python tests/test_engine.py
PYTHONPATH=src python tests/test_verifiers.py
PYTHONPATH=src python tests/test_cli.py
PYTHONPATH=src python tests/test_mcp_tools.py
PYTHONPATH=src python tests/test_canon_validators.py

All five suites should print All N tests passed. If any fail, open an issue with the output and your Python version.

3. Connect to Claude Desktop

Edit your Claude Desktop config:

  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows: %APPDATA%\Claude\claude_desktop_config.json
  • Linux: ~/.config/Claude/claude_desktop_config.json
{
  "mcpServers": {
    "concordance-engine": {
      "command": "concordance-mcp"
    }
  }
}

Restart Claude Desktop. The hammer / tool icon at the bottom of the chat shows "concordance-engine" with 14 tools available.

3. Connect to Claude Code (alternate)

claude mcp add concordance-engine -- concordance-mcp

4. Try it

In Claude Desktop:

"Use the concordance tool to check whether H2 + O2 -> H2O is balanced. If not, give me the correct coefficients."

Claude calls verify_chemistry, sees the imbalance, and replies with the balanced form.

Or for statistics:

"For a paired t-test with n=20, mean_diff=0.5, sd_diff=1.0, what's the two-tailed p-value?"

Claude calls verify_statistics_pvalue, gets the recomputed p, replies with the actual value rather than guessing.

Run from the CLI without an LLM

concordance validate examples/sample_packet_chemistry_verify.json --now-epoch 9999999999

The CLI runs the full Four-Gates pipeline against any JSON packet file. Exit codes: 0 PASS, 1 REJECT, 2 QUARANTINE, 3 schema invalid, 4 CLI error.

Troubleshooting

"command not found: concordance-mcp" — your virtualenv isn't activated, or you didn't install the [mcp] extra. Re-run pip install -e ".[mcp]" with the venv active.

"ModuleNotFoundError: scipy" — you installed the engine but not its dependencies. Run pip install -e ".[dev]" for the full set.

Tools not showing in Claude Desktop — fully quit Claude Desktop (not just close the window) and reopen. The MCP config is read at startup.
